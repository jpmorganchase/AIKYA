# Contributing to This Project

Thank you for your interest in contributing to this open source project! Currently, this repository is for reference-only purposes to provide verifiability of experiments conducted in the space of Federated Learning and its potential uses in financial applications.

**At this time, we are not accepting any contributions.**.
However, should this change in the future a note reflecting this will be added here.

## Important Notice

- All pull requests (PRs) raised will be **rejected or deleted**.
- This policy is temporary and may change in the future.
- Please do not submit PRs or changes at this time.
